발 측정 모듈(v2.0.0)

모듈이름-기능이름-번호(고려할 사항)

전체 전제 조건 : 본인의 폰으로 본인 발만 가능
유저 전제 조건 : 한국에 거주하고 한국식 이름이 있는 유저

1. 사진 촬영 기능(Take-TakeCMR-1)
전제조건 : 사용자의 휴대폰에서 카메라 촬영 기능이 정상적으로 작동됨, 카메라 사용권한을 설정할 수 있게 함
Take-TakeCMR-1 : 카메라의 기능을 이용하여 사용자가 촬영한다.

2. 발 사이즈 측정 기능(Take-Size-6)
전제조건 : 사용자가 튜토리얼을 확인했다.
Take-Size-1 : 사용자가 자신의 발, 검지발가락의 위치를 터치한 값을 포인터형 변수(Point FP)에 저장한다.
Take-Size-2 : 포인터형 변수의 값에서 y의 값을 사진의 끝(y좌표만)의 위치로 변경한 포인터형 변수(Point FF)에 저장한다.
Take-Size-3 : 포인터 FP와 포인터 FF의 값을 이은 선을 그리는 변수 line을 생성하여 계산식에 대입하고 DB에 저장한다.
Take-Size-4 : 사용자가 다음 버튼을 누른 후, 자신의 발 볼에서 가장 튀어나온 곳을 선택하면 포인터형 변수(Point fir)에 저장한다.
Take-Size-5 : 자신의 발 볼 끝을 선택하면 포인터형 변수(Point fin)에 저장한다.
Take-Size-6 : 포인터 변수 fir과 fin의 x좌표의 값을 이용하여 계산식에 넣어 계산해 DB에 저장한다.

3. 무지외반증 측정 기능(Take-Size-Cri-3)
전제조건 : 사용자가 튜토리얼을 확인했다, 포인터 변수의 값이 저장되어있다.
Take-Size-Cri-1 : 포인터 변수(Point FP,Point FF)의 값을 이용하여 두 점을 잇는 선을 생성한다.
Take-Size-Cri-2 : 포인터 변수(Point fir, Point FF)의 값을 이요하여 두 점을 잇는 선을 생성한다.
Take-Size-Cri-3 : 두 선을 이용하여 계산식에 대입하여 나오는 결과 값을 DB에 저장한다.
