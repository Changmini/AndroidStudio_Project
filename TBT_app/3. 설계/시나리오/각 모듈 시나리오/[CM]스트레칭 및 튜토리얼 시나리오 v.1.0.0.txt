19.10.06 (CM)이원식

스트레칭 시나리오
(외부 사이트에서 영상을 가져올 시)
Stretching_01 : 사용자가 원하는 스트레칭을 눌렀을 때 해당 버튼을 외부 사이트 영상 링크와 연결하여 동영상을 재생시킨다.

(직접 영상을 만들어서 사용할 시)
Stretching_01 : 사용자가 원하는 스트레칭을 눌렀을 떄 스트레칭 영상 페이지로 이동한다.
Stretching_02 : 스트레칭 영상 페이지에서 스트레칭에 대한 설명과 영상을 한번 보여준다.
Stretching_03 : 따라해보기 혹은 시작 버튼을 누를 시 영상 시작과 함께 타이머를 제공한다.
Stretching_04 : 해당 스트레칭이 끝나면 다음으로 넘어가 다음 스트레칭을 같은 방법으로 실시한다.

튜토리얼 시나리오
Tutorial_01 : 사용자가 회원가입 후 처음 로그인 하였을 때 본 앱을 처음 사용하는지 확인하는 창을 띄운다. (예, 아니요)
Tutorial_02-01 : 아니요 클릭 시 확인 창을 닫는다.
Tutorial_02-02 : 예 클릭 시 측정(발 상태) - 관리(스트레칭) - 신발 추천(쇼핑) 순으로 사용자에게 클릭을 유도하여 말풍선과 함께 설명을 시작한다.

- 앱의 중요한 기능만 설명, 마이메뉴에 대한 튜토리얼은 생략
